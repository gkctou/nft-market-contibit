EnftyMart

NFT Marketplace HTML Template.

EnftyMart is a HTML template which is based on latest Bootstrap 5 and its well optimized with mobile first responsive approach. EnftyMart is built using SASS preprocessor, HTML5, CSS3 and help of usefull npm plugins. The template is built with the latest Bootstrap 5 Version, making it completely free of jQuery dependency.

The html template uses Latest Bootstrap v5 with valid HTML5 tags. This template is a responisve layout with 4 column Support. All of the information in content area is nested within a class and comes with predefined classes.

Editing source scss and JS

In your package you will find gulpfile.js, package.json file and /src folder. Inside of /src folder included source of /scss, /js. So if you want to edit these files follow the instruction below.

  • Step 1 - open the command prompt window or terminal in your project folder.
  • Step 2 - Then run this command npm install, it may take a while to install.
  • Step 3 - After installation complete, you can run gulp develop command.
  • Step 4 - Now you can edit files from /src/scss, /src/js folder and it will compile automatically.
  • Step 5 - All the compiled files will store in /html/assets/css and /html/assets/js folder.
About to Install Node JS and Gulp

Download and install Node.js from nodejs.org/en/download/ (if your computer does't have it)

If you install Node JS for first time or you have not installed Gulp CLI before then run this command npm install --global gulp-cli.

Cautions! Please keep in mind that, we provide pre-compiled css and js file in your /html/assets folder. So now if you edit the source file from /src using gulp command then all the pre-compiled files will be replaced with newly compiled files.
As always before start editing, please backup your /html folder.
